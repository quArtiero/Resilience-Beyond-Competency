## ğŸš€ Deployment Guide

### Render Deployment

#### Prerequisites
1. Render account (https://render.com)
2. GitHub repository connected

#### Database Setup
1. Create PostgreSQL database on Render
2. Note the connection string

#### API Service Setup
1. Create Web Service for API
2. Build Command: `pip install -r requirements.txt`
3. Start Command: `cd api && alembic upgrade head && uvicorn app.main:app --host 0.0.0.0 --port $PORT`
4. Environment Variables:
```bash
DATABASE_URL=<your-postgres-url>
SECRET_KEY=<generate-secure-key>
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
PYTHON_VERSION=3.11.0
```

#### Frontend Setup
1. Create Static Site
2. Build Command: `cd web && npm install && npm run build`
3. Publish Directory: `web/dist`
4. Environment Variables:
```bash
VITE_API_URL=<your-api-url>
```

#### Deploy Content
```bash
# SSH into API service
python deploy_lesson26_standalone.py  # For EI Capstone
python seed_all_lessons.py           # For all lessons
```
